# Code of Conduct — AFRION Sahara

AFRION Sahara is building pan-African intelligence. That requires a serious, respectful community.

## Our standards
We expect:
- Respectful communication
- Constructive feedback and technical disagreement without personal attacks
- Inclusion across nationalities, languages, cultures, genders, and backgrounds
- Good-faith collaboration

We do not tolerate:
- Harassment, hate speech, threats, or doxxing
- Discriminatory language or behavior
- Sexual harassment or unwanted attention
- Sustained disruption, trolling, or bad-faith sabotage

## Scope
This Code of Conduct applies to:
- GitHub Issues, PRs, Discussions
- Any community spaces officially managed by AFRION Sahara

## Enforcement
AFRION maintains final authority on moderation decisions.
Actions may include:
- warning
- content removal
- temporary ban
- permanent ban
